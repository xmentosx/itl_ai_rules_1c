## Scope

Adds single-client adapters for Kimi Code, Qwen Code, Command Code, Cline, and
Pi alongside the existing five clients. Generic `other` and additive
multi-client installation remain unsupported.

Cline uses project `.cline/mcp.json`. Pi uses project `.pi/mcp.json` plus the
exact project package `npm:pi-mcp-extension@1.5.0`; the package integrity is
recorded in the adapter and host workflow dependency lock.

Publication requires the fork Full qualification, ten-client layout and
manifest coverage, MCP merge/removal regressions, publish `-WhatIf`, and
fresh-clone verification.
